# ES Spy

Spy on requests to ElasticSearch.

## Usage
	 dp-es-spy [-h|--help] [-a <Address>] <ESAddress> 
	  OPTIONS
		-h --help                 show help for action
		-a <Address>              Address to bind to (default: 127.0.0.1:9201)

	  ARGUMENTS
		<ESAddress>               Address of Elasticsearch host to connect
